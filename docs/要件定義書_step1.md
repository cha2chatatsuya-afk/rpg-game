# コンソールRPG 実装課題 ― フェーズ1：クラスを作る

## このお題のゴール

Javaで「クラスを1つ作って、それを使ってミニゲームを動かす」体験をします。
最終的には、勇者とモンスターがコンソール上で殴り合い、どちらかが倒れるまでのバトルが流れる、というところまで作ります。

このフェーズ1で身につく内容：

- フィールド（クラスが持つデータ）
- メソッド（クラスができる動作）
- コンストラクタ（インスタンスを作るときの初期化）
- インスタンス化（`new` でオブジェクトを作る）
- オブジェクトの参照（あるオブジェクトが別のオブジェクトを操作する）
- `while` ループでの繰り返し処理

> 継承・オーバーライド・抽象クラスは **次のフェーズ** で扱います。今回はまだ使いません。クラス1つだけに集中してください。

---

## 作るもの（完成イメージ）

実行すると、こんな感じでバトルが進みます（出力の文言は自由にアレンジしてOK）。

```
勇者 のHP: 30
スライム のHP: 15
----------------------------------------
勇者 の攻撃！ スライム に 8 のダメージ！
スライム のHP: 7
スライム の攻撃！ 勇者 に 4 のダメージ！
勇者 のHP: 26
----------------------------------------
勇者 の攻撃！ スライム に 8 のダメージ！
スライム のHP: -1
スライム は たおれた！
----------------------------------------
勇者 の勝ち！
```

このゴール画面を「完成の目印」にして実装を進めてください。

## クラス仕様：`Character`

ゲームに登場するキャラクター1体を表すクラスです。勇者もモンスターも、今回はこの1つのクラスから作ります。

### フィールド

| フィールド名 | 型 | 意味 |
|---|---|---|
| `name` | `String` | キャラの名前（例：「勇者」「スライム」） |
| `hp` | `int` | 体力。0以下になったら倒れたとみなす |
| `attackPower` | `int` | 攻撃力。攻撃時に相手のHPをこの値だけ減らす |

### コンストラクタ

```java
Character(String name, int hp, int attackPower)
```

- 受け取った3つの値を、それぞれ対応するフィールドに代入する。

### メソッド

#### `void attack(Character target)`
- **やること**：引数で受け取った相手（`target`）のHPを、自分の `attackPower` の分だけ減らす。あわせて「誰が誰に何ダメージ与えたか」をコンソールに表示する。
- **ヒント**：`target.hp` を書き換えます。「自分のフィールド」と「相手のフィールド」を取り違えないように注意。

#### `boolean isAlive()`
- **やること**：このキャラがまだ生きているか（`hp` が0より大きいか）を `true` / `false` で返す。
- 表示はしない。判定して返すだけ。

#### `void showStatus()`
- **やること**：このキャラの名前と現在のHPをコンソールに表示する（例：`勇者 のHP: 26`）。

> メソッドの「シグネチャ（名前・引数・戻り値）」は上記の通りに作ってください。**中身のロジックは自分で考えて実装する** のがこの課題の本番です。

---

## `main` の流れ

`Main` クラスの `main` メソッドに、ゲーム進行を書きます。

1. `Character` を2体作る（例：勇者 HP30/攻撃8、スライム HP15/攻撃4）。
2. 開始時の状態を両者とも表示する。
3. `while` ループで、**両者が生きている間** バトルを繰り返す：
   1. 勇者がスライムを攻撃する。
   2. スライムが倒れていたら、ループを抜ける。
   3. スライムが勇者を攻撃する。
4. ループを抜けたら、**生き残っているほう** を勝者として表示する。

> ループの止め方や、攻撃の順番の制御は自分で考えてみてください。`isAlive()` をどこで呼ぶかがポイントです。

---

## 実装の進め方（おすすめ手順）

一気に全部書こうとせず、小さく動かしながら進めると詰まりにくいです。

1. まず `Character` クラスのフィールドとコンストラクタだけ書く。
2. `main` でキャラを1体だけ作り、`showStatus()` で名前とHPが表示できることを確認する。
3. `attack()` を実装し、1回だけ攻撃させてHPが減ることを確認する。
4. `isAlive()` を実装する。
5. 最後に `while` ループでバトル全体をつなげる。

各ステップで「コンパイルして実行する」を挟むのがコツです。エラーは小さいうちに潰すほうが楽です。
